##環境構築に必要な手順
1.AWS Cloud9で新しくIDE(開発環境)を立ち上げる
2.AWS Cloud9の設定を行う
3.GitHubとの連携
4.GitHubの接続に必要な鍵の作成
5.作成した公開鍵をGitHubに登録
6.GitHubとの接続確認
7.Gitの設定を行う
8.Rails、ImageMagick等のセットアップ
*Railsのバージョンアップをする
